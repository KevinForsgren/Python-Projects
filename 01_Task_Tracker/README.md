# CLI- Based To-Do Manager

1. User can interact it with using CLI and can add, update and delete todo tasks as per convenience.

2. Task can be filter out on the basis of its status i.e:
    1. To-do
    1. In-progress
    1. Finished

3. User can store todo item with description and its status. This also record date and time when its created and updated. All todo items are distinquish with a unique id.

4. All data are stored in sqlite3 database for better security than .json or .txt file.

5. Set it up is very simple and anyone can just download this whole repo and start using it by creating the venv or using anaconda. 


---
If anyone wants to use venv they can paste the following command in the "To-Do_CLI-Based/" directory.
```
cd .\To-Do_CLI-Based\
```
and then,
```
python -m venv "your_env_name"
```
